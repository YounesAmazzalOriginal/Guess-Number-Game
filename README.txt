# Guess the Number Game

## Description
"Guess the Number Game" is a fun, interactive JavaScript game where players try to guess a randomly generated number within a specified range. The game features three difficulty levels (1-10, 1-50, and 1-100), and tracks the player's score. With each correct guess, the player's score increases, and the game displays the number of attempts made. The last score is saved and displayed, making it a great way to test and improve your JavaScript skills.

## Features
- Three difficulty levels: 1-10, 1-50, and 1-100.
- Real-time feedback with win/loss notifications.
- Tracks the number of guesses and displays the current score.
- Sleek interface with a responsive design for mobile devices.

## Technologies used
- HTML
- CSS
- JavaScript

## How to play
1. Select the difficulty level (1-10, 1-50, 1-100).
2. Enter your guess in the input box.
3. Click the "Valid" button to submit your guess.

4. If you guess correctly, you win and your score will increase. If not, you will be notified of your loss.

5. Your score and the number of attempts are displayed at the top and probability to win.

6. The last score is saved locally to track your best performance.
